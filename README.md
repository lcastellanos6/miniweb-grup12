## Com treballar-hi
Des de Git Bash:
1. Clona el repositori:

```bash
git clone  https://github.com/lcastellanos6/miniweb-grup12.git
```
2. Crea una branca amb el teu nom:
```bash
git checkout -b nom-alumne
```
3. Fes els teus canvis i puja'ls:
```bash
git add . 
git commit -m "Afegueixo part HTML/CSS"
git push origin nom-alumne
```
4. Crea un Pull Request a Github per fusionar els canvis.

Des de Visual Studio Code:
1. Anar a l'explorador d'arxius de VS Code i sel·leccionar l'opció de Clonar Repositori.
2. S'obrirà a la superior com un quadre de diàleg on podem sel·lecciónar la opció de clonar des de GitHub.
3. Es fa la sel·lecció del repositori que es vol clonar i s'obrirà una finestra de l'explorador d'arxius de l'equip per decidir on es guardarà la carpeta del repositori en local.
4. Un cop decidit al VS Code es port escollir si afegirla a l'àrea de treball, aquesta seria la opció per poder treballar i no haber d'obrir la carpeta constantment.
5. Has d'anar a la pestanya de Control de Codi Font de VS Code i polsar als tres punts (...) i buscar la opció de Rama. Una vegada trobada has de Crear una nova branca amb el teu nom i seguir les indicacions pertinents de VS Code.
6. Una vegada ja creada la nova branca ja es pot treballar.
7. per fer un commit amb el que s'hagi creat de nou o s'hagi modificat es fa a la pestanya de Control de Codi Font, escriure el missatge descriptiu del que s'ha realitzat des de l'ultima modificació fins al commit ques'està realitzant ara.
8. Ja es pot fer el commit, des de l'entorn gràfic no té molta pèrdua.
